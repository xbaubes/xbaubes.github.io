Chart.js és una llibreria JavaScript per crear gràfics interactius i visuals de manera senzilla.

  • Utilitza l’element <canvas> d’HTML
  • Permet crear gràfics de barres, línies, sectors, etc.
  • Funciona habitualment amb dades en format JSON
  • És ideal per dashboards i projectes web informatius

És molt utilitzada per mostrar dades de forma clara i atractiva en dashboards i projectes web informatius.

Aquest gràfic de barres mostra la distribució del nombre d’animals salvatges a Catalunya l’any 2023, ordenada alfabèticament per espècie.
Cada barra representa una espècie diferent i la seva alçada indica la quantitat estimada d’individus, utilitzant colors diferenciats per facilitar la lectura i comparació visual entre espècies.
