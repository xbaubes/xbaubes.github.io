Un timeline (o línia del temps) és una representació visual d’esdeveniments en ordre cronològic.
Permet veure com evolucionen fets, projectes o històries al llarg del temps, i facilita la comprensió de la seqüència i durada dels esdeveniments.

Els timelines s’utilitzen habitualment en contextos com la història, la gestió de projectes, l’anàlisi de processos o l’exposició de dades estadístiques.
Poden ser de dos tipus: estàtics (imatges, gràfics simples) o interactius (on es pot navegar, filtrar o ampliar informació).

TimelineJS és una biblioteca open source codificada amb JavaScript que permet crear timelines interactius i atractius per a pàgines web.
És fàcil d’integrar i permet incloure imatges, vídeos, mapes i text en cada esdeveniment.

Característiques principals de TimelineJS:

  • Totalment interactiu i responsive.
  • Permet afegir imatges, vídeos, mapes i text a cada esdeveniment.
  • Molt fàcil de configurar amb un fitxer JSON o Google Sheets.

Funcionament

Segueix les instruccions d’ús:
  1. Obre la plantilla amb les columnes incorporades, hauràs d'esborrar les files d'exemple.
  2. Fes clic a "Fes una còpia" per afegir aquest Google Sheets al teu Google Drive.
  3. Omple les columnes amb la informació de cada esdeveniment, per exemple:
    1. Data d'inici / Data de fi: Any, mes, dia.
    2. Títol: Nom de l'esdeveniment.
    3. Text: Descripció de l'esdeveniment.
    4. Media: URL d'imatge, vídeo o mapa.
    5. Tipus: "title", "era" o per defecte
  4. Publica el full de càlcul:
    1. Ves a Fitxer > Comparteix > Publicar al web.
    2. Fes clic a "Publica" i copia l'URL que apareix.
  5. Si no vols descarregar TimelineJS al teu servidor i, simplement, carregar els fitxers directament des del servidor oficial cada vegada que la pàgina s’obre,
    continua les instruccions que trobaràs a la web oficial per generar i inserir a la web el teu timeline interactiu.

Exemple: Història de Catalunya

Aquest timeline interactiu utilitza les llibreries de TimelineJS emmagatzemades localment, de manera que no depenem de servidors externs per carregar els fitxers CSS i JavaScript. Això garanteix que la línia del temps sempre funcioni encara que no hi hagi connexió a Internet o si el CDN oficial falla.

Per incloure les llibreries locals i utilitzar-les en la teva pàgina, consulta el codi d'aquesta pàgina descarregant-la des del repositori.
Si prefereixes utilitzar els servidors proporcionats per l'aplicació, només cal que incrustis el CDN que l'aplicació facilita.
Això permet carregar les llibreries de TimelineJS directament des del servidor oficial sense necessitat de tenir els fitxers localment.
Codi que podria usar per incrustar el timeline interactiu de "Història de Catalunya" usant CDN: 

	<iframe
		src="https://cdn.knightlab.com/libs/timeline3/latest/embed/index.html?source=v2%3A2PACX-1vQ8BiyXB5AXGWh6qQJ-EDVPQVlhRrXyQLloyjkZUqloRyc8PsQZu3UH7NANSMFKQMBm74mIkuqIoqHR&font=Default&lang=ca&initial_zoom=2&width=100%25&height=650"
		width="100%"
		height="650"
		frameborder="0"
		allowfullscreen>
	</iframe>

A continuació es mostra el Google Sheets publicat a la web des d'on TimelineJS obté les dades que utilitza per generar la línia del temps de "Història de Catalunya".
Cada fila de la fulla representa un esdeveniment amb la seva data, títol, descripció, imatges, etc.
Pots actualitzar el full de càlcul en qualsevol moment, i els canvis es reflectiran automàticament a la línia del temps.

Activitat per l'estudi dels àmbits ambiental, tecnològic o social

Escull un d'aquests temes per fer-ne una línia de temps amb TimelineJS:
  1. Eres i períodes geològics a la Terra
    • Per exemple, dins de l'era Mesozòica tenim:
      • Triàsic
      • Juràssic
      • Cretaci
    • Incloure espècies més comunes en cada període
    • Esdeveniments naturals clau
  2. Avenços científics/tecnològics i socials com:
    • Primera revolució agrícola
    • Primera revolució industrial
  3. Extinció d'espècies prehistòriques i modernes
    • Detallar-ne les causes
    • Legislació o acords de protecció de la biodiversitat
    • Protecció d'espècies en perill d'extinció
  4. Canvis climàtics i mediambientals al llarg de la història de la Terra com:
    • Glaciacions i períodes interglacials
    • Canvis en els nivells del mar
    • Desertificació

Requisits per el vostre TimelineJS:

  • Almenys 7 esdeveniments.
  • Per cada esdeveniment cal incloure, com a mínim:
    • Data d'inici i data de finalització.
    • Text explicatiu de l'esdeveniment.
    • Miniatura representativa.
    • Un element multimèdia (imatge o vídeo).
  • Incloure almenys 3 eres per agrupar els esdeveniments.
