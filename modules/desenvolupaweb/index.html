DESENVOLUPAMENT WEB

El desenvolupament web és el procés de creació i manteniment de llocs web. Això inclou una àmplia varietat de tasques, com ara el disseny de la interfície d'usuari (UI), la programació del codi que defineix el funcionament del lloc (amb HTML, CSS i JavaScript), la configuració de servidors, i la implementació de bases de dades. El desenvolupament web pot anar des de la creació de pàgines web simples fins a aplicacions web complexes i dinàmiques.

Els desenvolupadors web es poden especialitzar en diferents àrees, com ara el desenvolupament del front-end (tot allò amb què interactua l'usuari), el back-end (la lògica i les operacions que tenen lloc al servidor), o ser desenvolupadors full-stack, que cobreixen tant el front-end com el back-end.
